# eureka_odometry
eureka_odometry - пакет предназначенный для расчета одометрии по информации с энкодеров.

### Описание

Главный класс пакет EurekaOdometry имеет в себе нескольного функций способных обрабатывать сообщения типа ```sensor_msgs::msg::JointState```, которые несут в себе информацию о 
текущей скорости каждого колеса и его пройденный путь. На основе этой информации и некоторых параметров модели ровера (например, ширина следа ровера, длина корпуса) производится
подсчет пройденного пути - одометрии. Одометрия рассчитывается по принципу ```open_loop```, что означает передачу информации и ее последующую интеграцию в текущее состояние одометрии.

### Конфигурируемые параметры

Во избежание перекомпиляции пакета ```eureka_odometry``` в исходный код были добавлены конструкции позволяющие передавать параметры модели ровера и некоторые другие настройки
до запуска узла. Среди них:

- ```odometry_pub_topic: (string)``` - название топика, который будет публиковать одометрию в систему ровера.
- ```twist_pub_topic: (string)``` - название топика, который будет публиковать скорости из одометрии в систему ровера.
- ```joint_sub_topic: (string)``` - название топика на который будет подписываться данный узел для рассчета одометрии.
- ```imu_sub_topic: (string)``` - название топика на который будет подписываться данный узел для рассчета одометрии (опционально, если imu присутствует в системе).
- ```enable_odom_tf: (bool)``` - параметр, указывающий узлу необходимость публикации одометрии в виде tf дерева (публикует трансформацию odom -> base_link).
- ```wheel_radius: (double)``` - параметр модели ровера, указывающий узлу радиус колеса ровера (необходимый параметр для рассчета одометрии).
- ```wheel_base: (double)``` - параметр модели ровера, указывающий узлу длину корпуса ровера (необходимый параметр для рассчета одометрии).
- ```wheel_track: (double)``` - параметр модели ровера, указывающий узлу ширину следа ровера (необходимый параметр для рассчета одометрии).
- ```measure_error (double)``` - параметр, указывающий ошибку (неучетнных параметров ровера) при рассчете одометрии (подбирается эмпирически).

В скобках указаны типы значения которые принимают данные параметры.
